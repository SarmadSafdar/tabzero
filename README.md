# TabZero

AI-powered Chrome extension that gives support engineers full client context in one place. Zero tab switching.

## The Problem

Support engineers waste valuable time switching between multiple tabs on every single ticket. CRM, notes, infrastructure portals, ticketing systems, client documentation. TabZero solves this.

## What It Does

- Search any client and instantly load their full profile
- See VM and infrastructure status at a glance
- Describe the issue and AI analyzes it instantly
- Get likely cause, fix steps, and a ready-to-send client response
- One-click response templates for common situations
- Live ticket timer to track handle time

## How to Install

1. Download or clone this repo
2. Open Chrome and go to `chrome://extensions`
3. Enable Developer Mode (top right toggle)
4. Click "Load unpacked"
5. Select the TabZero folder
6. Done

## Setup

Open `popup.js` and replace `PASTE_YOUR_KEY_HERE` with your Groq API key. Get a free key at console.groq.com.
